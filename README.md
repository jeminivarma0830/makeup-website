# 💄 Makeup Website

A modern and visually appealing makeup website designed with a soft pink aesthetic. This project showcases beauty products, interactive UI, and a smooth user experience using web development technologies.

## ✨ Features

* 💖 Attractive and responsive design
* 🛍️ Product showcase section
* 👤 User name personalization
* 🎵 Background music integration
* ⏳ Animated effects (typing, loading, etc.)
* 📱 Mobile-friendly layout

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* (Optional) Flask for backend
